# DarkLord Remastered

**DarkLord Remastered** is a Discord multitool designed for advanced webhook and token-related operations. It provides multiple utilities for managing, modifying, and testing Discord webhooks. This project is fully open source.

<video src="[https://www.youtube.com/watch?v=G1WrZ7jzqP8](https://www.youtube.com/watch?v=G1WrZ7jzqP8)" controls width="100%"></video>

---

## Features

- **Webhook Spammer** – Rapid automated messages sent to a webhook.
- **Webhook Sender** – Send custom messages to a webhook.
- **Webhook Rename** – Change the display name of any webhook.
- **Webhook Edit Image** – Update the avatar of a webhook.
- **Webhook Info** – Display detailed information about a webhook.
- **Webhook Deleter** – Permanently delete a target webhook.
- **Token Generator** – Generate Discord tokens for testing and educational purposes.

---

## Installation

1. Ensure Python is installed on your system.
2. Download or clone the DarkLord Remastered project files.
3. Open your terminal or command prompt and navigate to the project directory.
4. Run the tool using:

```bash
python DarkLord.py
```

---

## Usage

After running `DarkLord.py`, you can access the features through the tool's interface. Each feature provides prompts for necessary input, such as webhook URLs, messages, or images.

**Example Commands:**

- **Webhook Spammer:** Enter a webhook URL and the message to spam.
- **Webhook Sender:** Enter a webhook URL and your custom message.
- **Webhook Rename:** Enter a webhook URL and the new name.
- **Webhook Edit Image:** Enter a webhook URL and image path.
- **Webhook Info:** Enter a webhook URL to view its details.
- **Webhook Deleter:** Enter a webhook URL to delete it.
- **Token Generator:** Follow prompts to generate tokens for testing.
